## Estrutura do projeto

projeto/
├── data/ # Todos os dados do projeto, separados por estágio do pipeline
│ ├── raw/ # Salve seus dados brutos, originais e imutáveis
│ ├── processed/ # Salve os dados já limpos e transformados
│ │ ├── v1_com_outliers/ # Salve os dados com limpeza geral, mas com os outliers mantidos
│ │ └── v2_outliers_tratado/ # Salve os dados com limpeza geral v1 + tratamento de outliers
│ └── final/ # Dados que serão usados futuramente em um treinamento de modelo de IA
│
├── notebooks/ # Notebooks para análise exploratória e experimentos
├── outputs/ # Salvar resultados finais, relatórios, e visualizações
│ ├── metricas_por_mes.csv # Resumo de métricas mensais extraído dos dados
│ ├── segmentacao_clientes.csv # Resultado da segmentação de clientes feita no notebook
│ ├── estatisticas_gerais.json # Resumo estatístico do dataframe
│ ├── graficos/ # Visualizações exportadas para análise e apresentações
├── README.md # Documentação do projeto, estrutura de pastas e justificativas de decisão
